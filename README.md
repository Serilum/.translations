# Serilum Mod Translations

![](https://workflow.serilum.com/badges/svg/translations.svg)

This repo contains the language files used in Serilum's mods that contain text elements. Collective manages these translations so each separate mod doesn't have to. This keeps everything organized and is a bit less work to maintain.

You can help translate into more languages, or fix existing translations, by going to:\
https://serilum.com/translations

A CloudFlare Pages worker is connected to the GitHub repo, which mirrors its files to:\
https://translations.serilum.com/

For example; you can find the manifest.json that's in this repo's root at:\
https://translations.serilum.com/manifest.json

If you're considering contributing some translations of your language, thank you so much!
